# Pensamento Computacional no Ensino Médio: Algoritmos Matemáticos

Este repositório contém uma proposta didática voltada para a integração da matemática com o pensamento computacional. O objetivo é capacitar alunos do Ensino Médio a interpretarem, modelarem e resolverem problemas matemáticos através da programação.

## 🎯 Objetivos do Projeto
* **Abstração:** Identificar conceitos matemáticos essenciais para a resolução de problemas.
* **Decomposição:** Dividir equações e fórmulas complexas em pequenos passos lógicos.
* **Reconhecimento de Padrões:** Aplicar lógica de repetição em progressões e funções.
* **Algoritmos:** Construir o passo a passo lógico (código) para automatizar cálculos.

## 🛠️ Ferramentas Utilizadas
* **Linguagem:** Python 3 (pela sua sintaxe próxima à linguagem humana).
* **Ambiente:** Google Colab ou VS Code.

## 📚 Conteúdos Abordados
Aqui estão alguns exemplos de algoritmos desenvolvidos:
1. **Álgebra:** Resolução de equações de 2º grau (Fórmula de Bhaskara).
2. **Sequências:** Cálculo de termos de uma Progressão Aritmética (PA) e Geométrica (PG).
3. **Financeira:** Simulador de Juros Simples e Compostos.
4. **Geometria:** Cálculo de áreas e volumes de figuras espaciais.
